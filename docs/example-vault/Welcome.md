# Welcome to your Alexandria library

This vault is your personal library of everything Alexandria has taught you.

## How it's organized

- One folder per project (here: [[Aurora/_index|Aurora]] and [[Atlas/_index|Atlas]]).
- Inside each project: `_index.md` lists every lesson, `_glossary.md` defines every concept, and `Sessions/` holds one note per lesson.
- `_Concepts/` holds ideas you've met in more than one project — like [[Idempotency]].

## Where to start

Open a project's `_index.md` and click any session. Concept names inside lessons link to the project glossary; glossary entries link back to every lesson that used them. The graph view shows the whole web.

*This is the hand-built example vault for schema v1 — its projects (Aurora, Atlas) and code references are illustrative samples, not real repositories.*
