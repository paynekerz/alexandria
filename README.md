# Alexandria

A suite of three Claude Code skills that teaches you your own codebases like a patient instructor, then archives every lesson into a personal Obsidian library — with glossaries, cross-project concept indexes, and drift detection.

**Status: pre-alpha. Under construction — nothing is installable yet.**

## The three skills

| Skill | Role |
|---|---|
| `alexandria-teach` | Explanation engine — reads your code and teaches it at your chosen depth |
| `alexandria-librarian` | Persistence — saves lessons into the Obsidian vault, maintains glossaries and indexes |
| `alexandria-recall` | Memory — surfaces past lessons before re-teaching, flags stale explanations |
